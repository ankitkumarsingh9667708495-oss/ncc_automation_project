# ncc_automation_project

A web-based NCC management system developed to digitize and automate administrative tasks within National Cadet Corps units. The application enables efficient management of cadet records, attendance, events, camps, achievements, and announcements through a role-based access system for Administrators, Officers, and Cadets.

Features
Cadet registration and profile management
Attendance tracking and reporting
Event and camp management
Achievement records and performance monitoring
Announcement and notification system
Secure authentication and role-based access control
Tech Stack
Backend: Django (Python)
Frontend: HTML, CSS, Bootstrap, JavaScript
Database: MySQL / SQLite
Objective

To reduce manual paperwork, improve transparency, and streamline NCC administrative operations through a centralized digital platform.
